MARKDOWN FILTER FORMAT by Ceardach

A feature module based on the Kit specifications (http://drupal.org/project/kit)

Provides the following functionality:

  * Filter format 'markdown'
    - Allows Anonymous and authenticated users
    - Provides the following filters:
      * Limits allowed HTML to:
        <a> <em> <strong> <cite> <blockquote> <code> <ul> <ol> <li> <dl> <dt>
        <dd> <img> <br>
      * Code filter
      * Markdown
      * Convert URL into links
      * Typogrify
    - Configures typogrify


Required contributed modules:

  * features
  * markdown
  * codefilter
  * typogrify
